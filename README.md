# Project Management System

A backend Project Management System built with **Django REST Framework** featuring **JWT Authentication**, a **Custom User Model**, and **Role-Based Access Control**.

## Features

- Custom User Model
- JWT Authentication
- User Registration
- User Login
- Protected Profile API
- Role-Based User Structure (Admin & Employee)
- Custom Permission Classes

## Tech Stack

- Python
- Django
- Django REST Framework
- Simple JWT
- SQLite

## Run the Project

```bash
git clone <repository-url>

cd Project_management_system

python -m venv .venv

# Activate virtual environment

pip install -r requirements.txt

python manage.py migrate

python manage.py createsuperuser

python manage.py runserver
```
